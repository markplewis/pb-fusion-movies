## Startup

1. Run `npm install`
2. Start your "Docker Desktop" app
3. Run `npx fusion start`
4. Go to PB admin: http://localhost/pf/admin/app/browse/pages.html
5. Run database migrations (if applicable)
6. Paste the required data into the `.env` file
